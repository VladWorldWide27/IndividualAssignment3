Projects

Cleanup Crew

This project is a 2D Rust game that procedurally generates space-station rooms, enemies, and hazards so each run feels different. It also simulates realistic gas flow, decompression, and object movement in destructible environments, while tying everything together with combat, progression, and dynamic difficulty systems.

BetterPath: Accessible Navigation on Campus

This project is an accessibility-focused navigation app that helps users plan safer campus routes by uploading photos of hazards and improving route and ETA accuracy. It uses a FastAPI and MongoDB backend with graph-based pathfinding, plus Mapbox for live maps and Gemini for validating reported accessibility obstacles.
